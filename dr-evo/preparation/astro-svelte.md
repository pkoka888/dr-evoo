## dr-evo.cz Specific Recommendations

Based on the "Vibe Coding" mission (artisan wood crafter site):

### Recommended Stack
- **Framework**: Astro 6.0 (static ship, excellent SEO)
- **Interactivity**: Svelte 5 with Runes ($state, $derived)
- **Styling**: Tailwind CSS with custom artisan theme
- **Backend**: Payload CMS 3.0 for order management

### Key Repositories to Reference
1. **Astrowind** - Tailwind styling patterns
2. **fulldotdev/ui** - shadcn-compatible components
3. **Accessible Astro Starter** - WCAG compliance patterns
4. **kampsy/ui** - Svelte 5 components (Geist-inspired)
5. **Payload CMS** - Order backend

### Integration Notes
- Use `client:visible` for Svelte "islands" in Astro
- Use Nano Stores for shared state between Astro and Svelte
- Zod for Czech phone/email validation (+420 format)
- Use Payload CMS MCP server for AI order processing
