# Dr-Evo Preparation Directory

> Research index and organized findings for AI agents, skills, rules, workflows, MCP, and knowledge base

## Purpose

This directory stores research results from the `/research` workflow. Findings are organized by category and can be used as reference for building agent capabilities.

## Directory Structure

```
preparation/
├── index.md              # Main index with all categories
├── agents/               # Agent templates and frameworks
├── skills/              # Skills and knowledge modules
├── rules/               # Rules and guidelines
├── workflows/           # Workflows and automation
├── mcp/                 # MCP servers and integrations
└── knowledge-base/       # Knowledge management systems
```

## Usage

Research is executed via the `/research` workflow:

```bash
/research all          # Research all categories
/research agents      # Research agent frameworks
/research skills      # Research skills
/research mcp         # Research MCP servers
```

## Index Format

Each category directory contains markdown files with:
- Repository name and URL
- Stars and language
- Description and use cases
- Categories and tags
- Integration points

## Maintenance

Run `/research` periodically to:
- Add new discoveries
- Update existing entries
- Refresh category indexes
