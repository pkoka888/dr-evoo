Kilo Code supports an extensive range of AI providers, giving you the flexibility to choose the best model for each task. Whether you need the latest GPT model, Anthropic's Claude, or want to run local models, we've got you covered.

**Provider support:**
- OpenAI (GPT-4, o1, o3)
- Anthropic (Claude 3.5, 3.7)
- Google (Gemini 1.5, 2.0)
- OpenRouter aggregations
- Local models via Ollama
- Custom provider endpoints
