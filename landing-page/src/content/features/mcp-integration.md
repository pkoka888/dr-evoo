Kilo Code's MCP integration lets your AI assistant connect directly to external tools and services. Query databases, call APIs, and interact with development tools without leaving your editor.

**MCP capabilities:**
- Database connections (PostgreSQL, MySQL, MongoDB)
- API integrations (REST, GraphQL)
- Cloud service connections (AWS, GCP, Azure)
- Development tools (GitHub, Jira, Slack)
- Custom MCP server support
- Secure credential management
