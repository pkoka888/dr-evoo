# 📋 Project Plan: Repo Integrations

This document outlines the plan for integrating external repositories and services into the dr‑evo project.

## Goals
- Define how cloned research repositories (`.repos‑studies/`) are managed.
- Set up CI/CD pipelines for automated sync of study data.
- Establish conventions for naming and versioning of integration artefacts.

## Milestones
1. **Initial Setup** – Add `.repos‑studies/` to `.gitignore` (already done).
2. **Sync Automation** – Create a script to pull updates from upstream research repos.
3. **Documentation** – Keep each study in `plans/studies/` with consistent front‑matter.

*This file is a living document and will be expanded as integration work progresses.*
